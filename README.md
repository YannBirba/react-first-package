# react-first-package

## Npm

### Global link

```bash
npm link
```

### Use the link

```bash
npm install react-first-package
```

### Unlink

```bash
npm unlink
```

### Publish

```bash
npm publish
```

### Version patch

```bash
npm version patch
```

### Version minor

```bash
npm version minor
```

### Version major

```bash
npm version major
```
